# AIstronaut (Python Edition)

This is a Python (Pygame) port of the AIstronaut game, specifically adapted for larger screens like MacBooks or Windows laptops.
It uses a "fog" margin effect when played on wider aspect ratios to preserve the original mobile-like difficulty and perspective.

## Requirements
- Python 3.7+

## Installation

```bash
pip install -r requirements.txt
```

## Running the Game

```bash
python main.py
```

## Controls
- **SPACE** or **Left Mouse Click**: Tap and hold to thrust. Release to drift.
