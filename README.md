## Download Android App
You can download the latest Android APK from the **[GitHub Actions](https://github.com/chubarevgeniy/AIstronaut-/actions)** tab:
1. Go to the "Actions" tab.
2. Click on the latest "Android Build" workflow run.
3. Scroll down to the "Artifacts" section.
4. Download the `app-debug` zip file.

## Mission Briefing
Your goal is to fly as high as possible away from Earth.
- **Controls**: Tap and hold to thrust. Release to drift.
- **Physics**: Use gravity assists from planets to conserve fuel and gain speed.
- **Modes**:
    - **Survival**: Limited fuel. Use it wisely.
    - **Zen**: Infinite fuel. Just fly.

## Tech Stack
